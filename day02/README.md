## 变量

### 定义

- 动态语言, 不需要预先声明变量的类型
- 变量类型由它的值决定
- 变量在使用之前必须先赋值

### 命名约定

- 第一个字符只能是大小写字母或下划线
- 后续字符只能是: 大小写字母/数字/下划线
- 区分大小写
- 有意义的名称
- 类名: 驼峰命名: MyClass
- 变量使用名词, 函数使用**谓词**(动词+名词)

### 变量操作

- 变量类型和值在赋值的那一刻被初始化
