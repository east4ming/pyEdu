# Python 企业开发

## 数据类型与操作

- 整形
- 字符串
    - 字符串拼接： 通过“+”号
    - 字符串填充： 通过“%”或`fromat()`方法
    - 多行注释： 3个引号`""""""` `''''''`
- 布尔
    - True
    - False
- 列表： 有序集合
    - 长度： `len()`
    - 元素可以不是同一类型， 可以内嵌list
    - 切片
    - 列表元素追加：`append()`
    - 列表拓展： `extend(list)`
    - 删除： `pop(index)` `remove(value)`
    - 修改元素：索引赋值
- None：
    - 空的列表/元组
    - 整数： 0
    - 空的字典/集合
