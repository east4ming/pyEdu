## 文件

- r: 读(不存在则报错)
- w: 写(存在则清空, 不存在则创建)
- a: 追加(必要时创建新文件)
- r+: 读写方式(参见r)
- w+: (参加w)
- a+: 参加a
- b: 二进制

### 文件操作3步骤

- 打开
- 读写(建议`f.read(4096)`)
- 关闭

### 读取

- read
- readline - 读取一行数据(包括`\n`)
- readlines - 读取所有数据,每一行为列表的一个元素
- 循环遍历

### 写入

- write
- writelines -  列表
- flush - 立即将缓存中的数据写入硬盘

### `with open() as...:`
